## ✈ 

**Воздушный бой** — это многопользовательская браузерная игра в жанре «стрелялка по координатам», вдохновлённая классическим морским боем, но перенесённая в полноценное **3D-пространство**.

Вместо кораблей на плоской сетке — эскадрильи самолётов в кубе 10×10×10.  
Вместо строк и столбцов — три оси X, Y, Z.  
Вместо скуки — тактика, пространственное мышление и адреналин.

---

## 🚀 Запуск

```bash
# Склонировуй репозиторий
git clone https://github.com/your-username/aerial-combat.git
cd aerial-combat

# Установи зависимости
npm install

# Запустить сервер
node server.js

# Гойда!
```

После запуска появятся ссылки для двух игроков.
Перейди по ним со своим другом и начнется Воздушный бой!
